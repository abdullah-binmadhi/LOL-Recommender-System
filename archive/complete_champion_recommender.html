üß† Machine Learning Algorithm

Weighted Scoring System

Our algorithm uses a multi-criteria decision analysis (MCDA) approach with weighted scoring:

Step 1: Role Matching (Weight: 40 points)

Perfect role match receives maximum points. This is the highest weighted factor as role preference is fundamental to gameplay enjoyment.

if (userRole === championRole) score += 40
Step 2: Difficulty Alignment (Weight: 4x multiplier)

Uses inverse distance scoring to match user skill level with champion complexity:

difficultyScore = max(0, 10 - |userDifficulty - championDifficulty|) √ó 4
Step 3: Attribute Matching (Weighted by importance)
  • Damage & Toughness: 3x multiplier (core gameplay factors)
  • Control, Mobility, Utility: 2x multiplier (secondary factors)
attributeScore = max(0, 10 - |userPreference - championAttribute|) √ó weight
Step 4: Playstyle Bonuses (+10 points)

Additional scoring for playstyle alignment:

  • Aggressive: Bonus for champions with damage ‚â• 8
  • Defensive: Bonus for champions with toughness ‚â• 7
  • Supportive: Bonus for champions with utility ‚â• 8

üìà Confidence Score Calculation

The confidence percentage represents how well a champion matches your preferences:

Confidence = min(95, max(65, round(totalScore / 2)))
  • 85-95%: Excellent match - highly recommended
  • 75-84%: Very good match - strong recommendation

üî¨ Technical Implementation

Core Algorithm (JavaScript)

function getRecommendation(answers) {
    const scores = {};
    
    for (const [name, champion] of Object.entries(allChampions)) {
        let score = 0;
        
        // Role matching (40 points)
        if (answers[1] === champion.role) score += 40;
        
        // Difficulty matching (weighted)
        const difficultyScore = Math.max(0, 10 - Math.abs(answers[2] - champion.difficulty));
        score += difficultyScore * 4;
        
        // Attribute matching (weighted)
        score += Math.max(0, 10 - Math.abs(answers[3] - champion.damage)) * 3;
        score += Math.max(0, 10 - Math.abs(answers[4] - champion.toughness)) * 3;
        score += Math.max(0, 10 - Math.abs(answers[5] - champion.control)) * 2;
        score += Math.max(0, 10 - Math.abs(answers[6] - champion.mobility)) * 2;
        score += Math.max(0, 10 - Math.abs(answers[7] - champion.utility)) * 2;
        
        // Playstyle bonuses
        const playstyle = answers[8];
        if (playstyle === 'aggressive' && champion.damage >= 8) score += 10;
        if (playstyle === 'defensive' && champion.toughness >= 7) score += 10;
        if (playstyle === 'supportive' && champion.utility >= 8) score += 10;
        
        scores[name] = score;
    }
    
    return Object.entries(scores).sort(([,a], [,b]) => b - a);
}
