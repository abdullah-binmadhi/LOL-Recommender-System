# Documentation Archive

This folder contains development documentation, task summaries, and test files that were created during the project development phase.

## Contents

- **DEPLOYMENT_*.md** - Deployment and monitoring documentation
- **FINAL_*.md** - Final implementation reports
- **TASK_*.md** - Individual task completion summaries
- **ML_*.md** - Machine learning implementation documentation
- **TEST_*.md** - Test results and summaries
- **UNIFIED_*.md** - ML unification design documents
- **test_dataset_sample.*** - Sample test datasets
- **LoL_champion_data.csv** - Original champion data (backup)

## Note

These files are archived for reference but are not needed for the application to run. The live application uses data from `/src/data/` folder.

For current documentation, see:
- `/README.md` - Project overview
- `/USAGE_GUIDE.md` - User guide
- `/docs/` - GitHub Pages documentation
